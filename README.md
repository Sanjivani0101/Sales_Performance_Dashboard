# Sales_Performance_Dashboard
The project focuses on analyzing sales performance, manufacturer contribution, category-wise revenue, and year-over-year business growth using Power BI.

The objective was to transform raw sales data into actionable business insights by performing data cleaning, data modeling, DAX calculations, and interactive dashboard development.

Manufacturer Analysis: Monitored manufacturer revenue, growth trends, and market contribution to identify top-performing business units.

Sales Analysis: Analyzed sales trends, category performance, and growth metrics to support data-driven business decisions.


The dataset files are not included in this repository due to GitHub file size limitations.
